Проект: XYear
Описание:
- Показывает текущую дату, дату ближайшего Нового года (1 января следующего года) и сколько дней осталось.
- Тёмная тема включена по умолчанию.
- Авто-обновление при старте и каждые 60 секунд.
Сборка:
- Откройте файл DaysToNewYear.csproj в Visual Studio (поддерживаются .NET 8.0 и Windows Forms).
- Нужен .NET SDK 8.0 и Windows
